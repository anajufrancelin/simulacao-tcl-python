# Simulação do Teorema Central do Limite
Projeto acadêmico desenvolvido com o objetivo de demonstrar, por meio de simulações computacionais, o funcionamento do Teorema Central do Limite (TCL).
O trabalho utiliza Python para gerar amostras aleatórias, calcular médias amostrais e analisar o comportamento das distribuições obtidas, permitindo observar a convergência para uma distribuição aproximadamente normal conforme o tamanho da amostra aumenta.

# Objetivos
- Compreender o Teorema Central do Limite na prática.
- Aplicar conceitos de Probabilidade e Estatística.
- Realizar simulações computacionais utilizando Python.
- Interpretar resultados por meio de gráficos e análises estatísticas.

# Tecnologias utilizadas
- Python
- NumPy
- Matplotlib
- Estatística Descritiva

# Funcionalidades
- Geração de amostras aleatórias.
- Cálculo de médias amostrais.
- Construção de histogramas.
- Análise da distribuição dos resultados.
- Visualização gráfica dos experimentos.

# Resultados
A simulação demonstrou que, independentemente da distribuição original dos dados, a distribuição das médias amostrais tende a se aproximar de uma distribuição normal à medida que o tamanho da amostra aumenta.
